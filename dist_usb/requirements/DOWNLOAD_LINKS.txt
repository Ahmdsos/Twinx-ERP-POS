   Twinx ERP — Required Software Downloads
═══════════════════════════════════════════════════

You need to install ONLY ONE program: XAMPP
Everything else is included in this package.

───────────────────────────────────────────────────
  ★ XAMPP (REQUIRED)
───────────────────────────────────────────────────
  Contains: Apache + MySQL + PHP 8.2+
  
  Download Link:
  https://www.apachefriends.org/download.html

  ⚠ Choose the version with PHP 8.2 or higher!
  
  Installation:
  1. Run the installer
  2. Use default path: C:\xampp
  3. Select: Apache, MySQL, PHP
  4. Click Install

───────────────────────────────────────────────────
  ★ After installing XAMPP:
───────────────────────────────────────────────────
  1. Close this file
  2. Double-click SETUP.bat
  3. Everything else is automatic!
